# Minimal Vocabulary

This is a compact vocabulary for Codex and agent use. It is intentionally small. Do not turn this repo into a large tag library.

Use these terms when a prompt needs clearer camera, lighting, composition, mood, action, or look language.

## Camera

| Term | Use When |
|---|---|
| full-body shot | The whole character must be visible. Pair with "head-to-foot visible". |
| medium-full shot | The character should be readable from face to body action without becoming tiny. |
| close-up | Face or detail is the focus. Do not combine with full-body unless using panels. |
| three-quarter front | Useful for character cards and readable costume design. |
| low angle | Makes the character feel stronger, sacred, or dramatic. |
| eye-level | Neutral and clear. Useful for character sheets. |
| slight overhead | Useful for shrine floors, seals, maps, or ground interaction. |
| diagonal composition | Use when gaze, hand, and object should form a story line. |

## Lighting

| Term | Use When |
|---|---|
| primary light source | Define the main light before secondary glow or effects. |
| cool moonlight | Night, sacred, quiet, or solemn scenes. |
| soft overcast light | Clean character sheet or gentle outdoor scene. |
| rim light | Separate the silhouette from a dark background. |
| restrained sacred glow | Symbolic effect that must not overpower the character. |
| warm lantern light | Interior shrine, old street, or intimate story moment. |
| lighting hierarchy | Use when multiple lights exist; state which one controls the form. |

## Composition

| Term | Use When |
|---|---|
| centered character card | Clear mobile-game card or deity-card presentation. |
| off-center subject | Narrative scene with space for background story elements. |
| head-to-foot visible | Full-body output where feet must not be cropped. |
| clean negative space | Prevent clutter around the character. |
| panel grid | Character sheet only. Do not use for single-scene illustration. |
| clear panel spacing | Prevent character sheet panel blending. |
| foreground object | Adds depth, but must not hide face or hands. |
| background supports subject | Prevents environment takeover. |

## Mood

| Term | Use When |
|---|---|
| solemn | Sacred, formal, or ritual image. |
| protective | Healer, guardian, or deity-like character. |
| quiet alarm | Discovery scene without exaggerated action. |
| restrained awe | Mythic moment with controlled effects. |
| tense calm | Narrative moment before action escalates. |
| gentle warmth | Healing, refuge, or intimate character moment. |

## Action And Look

| Term | Use When |
|---|---|
| reaches toward | Makes a story object or clue active. |
| steps forward | Adds motion while preserving a clear full-body pose. |
| turns back | Creates story tension or interrupted movement. |
| kneels beside | Useful for shrine seals, injured characters, or discovered objects. |
| raises one hand | Ritual gesture, blessing, or spell action. |
| wind moves fabric | Adds life without changing identity. |
| controlled detail density | Keeps the image polished but readable. |
| mobile-readable silhouette | Useful for game card art. |

## Usage Rules

- Use vocabulary to clarify a field, not to pad the prompt.
- Prefer one strong camera phrase over several conflicting ones.
- Prefer one primary light source over many equal light sources.
- Use action verbs for `narrative_scene`; atmosphere alone is not enough.
- Keep character identity terms outside motion-heavy sentences when strict consistency matters.
