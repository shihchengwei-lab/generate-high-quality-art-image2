# Core Principles

This repo is a general-purpose high-quality Image 2.0 skill.

Principles:

- Host-native `image_gen` is the generation path.
- Local scripts validate specs and export optional diagnostics only.
- The v2 contract is the only runtime contract.
- Reference images are limited authority sources.
- Preserve / Change / Ignore is required before visual description.
- Quality preflight happens before generation.
- Diagnostics do not prove visual quality and do not replace post-generation review.
- No external provider, API fallback, batch path, or hidden compatibility layer is provided.
