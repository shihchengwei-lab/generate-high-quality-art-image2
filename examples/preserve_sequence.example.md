# Preserve Sequence Example

Task:

- Build prompt-planning files for related outputs with a stable object canon.

Sequence contract:

- Preserve canon: same ceramic lantern form, same glaze material, same clean finish.
- Allowed variation: environment, time of day, camera distance.
- Forbidden variation: changing silhouette, replacing material, adding visible text or logos.

Expected output:

- `sequence_guide.md`
- `variation_matrix.md`
- one prompt file per image
- `sequence_summary.md`
