# Prompt Structure

Prompt assembly order:

1. Goal, `task_type`, `image_type`, and `intended_use`.
2. Reference authority and role assignments.
3. Preserve / Change / Ignore.
4. Reference contamination guards.
5. Main subject.
6. Composition, camera, crop, and arrangement.
7. Scene, lighting, and atmosphere from user text.
8. Rendering style.
9. Visual accuracy and clean-render contract.
10. Output constraints.
11. Quality checks.
12. Avoid list or selected negative modules.

Keep the Preserve / Change / Ignore block before concrete scene description. This prevents source details from taking over before the contract is stated.
