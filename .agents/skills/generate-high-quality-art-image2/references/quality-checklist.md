# Quality Checklist

After generation, review:

- requested subject, object, action, output form, and change scope
- reference roles used only within declared authority
- Preserve / Change / Ignore honored
- composition, camera, crop, spatial arrangement, and style fit
- visible bodies, hands, objects, materials, and props are coherent
- lighting, scene, atmosphere, and effects follow user text
- clean edges, stable gradients, controlled texture, and no dirty haze
- no random text, logos, watermarks, UI, unrelated props, or extra subjects unless requested

If diagnostics are requested, revise one visible failure while keeping the original contract.
