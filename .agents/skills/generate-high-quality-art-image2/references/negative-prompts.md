# Negative Prompt Modules

Use negative prompts as risk controls after the positive contract is clear.

Required principle:

- State what to preserve and change positively first.
- Use negative modules only to block artifacts, contamination, or unwanted extras.

Risk families:

- render cleanliness: noise, dirty texture, unstable edges, broken gradients
- body anatomy: malformed hands, broken joints, impossible posture
- object/material complexity: fragmented objects, cluttered props, unstable materials
- lighting/effects noise: scattered glow, halos, unreadable effects
- environment/background control: clutter, incoherent structures, unwanted symbols
- text artifacts: random text, labels, logos, watermarks, UI, code fragments
