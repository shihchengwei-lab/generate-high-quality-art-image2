# Wishwalking Style Bible

This file defines the preferred art direction for the game world "願行".

Use this style bible only when the user asks for Wishwalking / 願行 / Taiwanese folk‑belief game art, deity cards, divine scenes, shrine atmosphere, blessing moments, or related visual assets.

## Core mood

The image should feel:

- warm
- sacred
- quiet
- emotionally grounded
- compassionate
- spiritual but not horror
- mythic but not generic fantasy
- intimate rather than spectacular
- suitable for a mobile game with a folk‑belief atmosphere

## Visual language

Preferred elements:

- Taiwanese folk‑belief atmosphere
- incense warmth
- amber and soft gold light
- quiet blessing moment
- gentle divine presence
- readable silhouettes
- restrained symbolic detail
- soft temple or village atmosphere
- human emotional grounding
- clean 2D illustration
- controlled glow
- calm sacred light
- subtle material richness

Avoid:

- horror temple atmosphere
- cyberpunk neon
- generic fantasy idol look
- fake Chinese text
- random glyphs
- random code fragments
- excessive digital magic circles
- overdesigned costume clutter
- unreadable ornaments
- cold metallic sci‑fi lighting
- overly glossy plastic surfaces
- chaotic talisman-like symbols unless specifically requested

## Character art rule

The character should feel like a living deity, spiritually significant person, shrine guardian, or blessed figure within the world of the game.

Do not make the image feel like:

- a generic anime fantasy idol
- a gacha splash screen with chaotic effects
- a horror deity
- a cyberpunk spirit
- a Western high‑fantasy god
- a random temple cosplay image

## Detail hierarchy

High quality does not mean more details everywhere.

Use this detail priority:

1. face and expression
2. hands and blessing gesture
3. silhouette and costume hierarchy
4. symbolic costume details
5. sacred light
6. background atmosphere

## Lighting rule

Use warm sacred light, incense glow, soft amber highlights, and controlled divine atmosphere.

Avoid uncontrolled glitter, harsh rim halos, noisy particles, snow-like sparkle over the subject, and random glowing symbols.

## Costume rule

Costume should have a clear hierarchy:

- primary robe shape
- secondary fabric layers
- symbolic accent
- restrained ornament
- readable color blocks

Avoid fragmented robe panels, excessive ribbons, random ornaments, chaotic accessories, unreadable pattern density, and too many competing decorative elements.

## Background rule

Background should support the character.

Good backgrounds:

- soft temple atmosphere
- warm shrine light
- abstract incense haze
- village dusk
- festival lantern atmosphere
- muted sacred interior
- quiet spiritual space

Avoid backgrounds that compete with the subject or contain fake text, random glyphs, dense ornaments, broken symbols, or noisy signage.

## Mobile readability

For mobile game use, the image should remain readable when reduced in size.

Prioritize:

- clear face
- clear gesture
- clear silhouette
- separated subject/background values
- non‑chaotic lighting
- controlled detail density
